Evaluation of Studio 4 Design

One priority to tie in all the visual elements together as part of one, single user interface was to make sure all the lements maintained a consistent visual design. One of the ways in which I did this was to use a popping color scheme, and make sure all the elements correlated with the rest. Blog Post #3 2/7/17

Evaluation of Studio 3 Design

I edited the image so that the interactive elements would be visually different from the rest of the image. I chose to desaturate each of the 6 bowling balls in order to indicate that there was something "missing" from them, and so the user would hover over each one to find out more information. Thus, I left out a CTA. Three ways in which this is an effective and interesting interface design is that there is no CTA (the user should be able to find out on his/her own), the images are all custom-shaped (no circles or rectangles), and lastly the videos are all custom-shot as well.

The design of the form is extremely straightforward, although it is aesthetically also very plain and doesn't have much going on. However, it is still informative and does it purpose.

Blog Post #1 1/12/17

I ultimately chose Reddit as my choice of interface design to evaluate. The reason I chose Reddit was because Reddit is one of the most popular interfaces browsed around the world, and it features many different types of users. However, all users have one purpose on the site – they seek information and so they want to learn.


Design: The design of Reddit is straightforward, though it is definitely outdated. The white background is meant to enable users to read text easily, and the lack of imagery and animations in general enables users to load the page as quickly as possible. This goes hand-in-hand with Reddit being a text-heavy site. Users come to the site to converse with each other, read each other’s comments, and write their own. The consistency and the spacing of the website make the site very easy to use.


Appeal: To put it bluntly, Reddit relies on its content for appeal to users, not its aesthetic appeal. Nevertheless, thousands of users visit the site every hour to refresh themselves on the news and to read each other’s opinions on countless topics.


Effectiveness: Ultimately, type hierarchy and consistent spacing are the main reasons why Reddit is able to function as a heavy-duty website. However, I would not rate it as ‘effective’ as it is easy to drown oneself in all of the content that it has to offer, as there are nearly a hundred clickable links but only 3 different colors for the links.
